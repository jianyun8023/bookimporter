### 2. 环境变量章节

```markdown
## 🔧 环境配置

### 颜色输出控制
\`\`\`bash
# 禁用彩色输出（适用于 CI/CD 环境）
export NO_COLOR=1

# 确保 UTF-8 支持
export LANG=en_US.UTF-8
\`\`\`

### 终端兼容性
BookImporter 会自动检测终端能力：
- 支持彩色终端时使用彩色输出
- 不支持时自动降级为纯文本
- Unicode 图标自动降级为 ASCII
```

## 🛠️ 技术栈

- **语言**: Go 1.18+
